# Solum Design System

Components generated from the Figma library [**Component (Shared)**](https://www.figma.com/design/YqKny45xSmjr76WGIXeL7P/Component--Shared-?node-id=51-158).

## Categories in Figma

| Section | Storybook path |
|---------|----------------|
| Button | Controls/Button |
| Checkbox | Controls/Checkbox |
| Badge | Status Indicator/Badge |
| Input | Input/Input |
| Charts | Charts/Horizontal Bar Chart |

## Design tokens

Tokens live in `src/styles/tokens.css` and map to Figma variables (Pretendard, primary `#282363`, semantic colors, spacing, radius).

## Run locally

```bash
npm install
npm run storybook
```
